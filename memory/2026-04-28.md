### Warnung: Trusted Proxies
**Befund:** Reverse proxy headers sind nicht vertraut  
**Grund:** `gateway.bind` ist loopback (127.0.0.1) und `gateway.trustedProxies` ist leer  
**Risiko:** Wenn du die Control UI über einen Reverse Proxy freigibst, könnten local-client checks gespooft werden  
**Fix:** Entweder `gateway.trustedProxies` mit deiner Proxy-IP konfigurieren, oder Control UI nur local-only nutzen
