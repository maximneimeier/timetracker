### Warnung: Reverse Proxy Headers
**Problem:** `gateway.trustedProxies` ist leer, obwohl Gateway über Loopback gebunden ist.
**Impact:** Wenn das Control UI über einen Reverse Proxy erreichbar gemacht wird, können Local-Client-Checks gespooft werden.
**Empfehlung:** Entweder `gateway.trustedProxies` auf Proxy-IPs setzen ODER Control UI lokal-only behalten.
