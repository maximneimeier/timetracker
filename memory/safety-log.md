# Safety Log

Diese Datei protokolliert Safety-relevante Events.

## Format

```markdown
## TIMESTAMP
**STATUS** - Event Type
Details...

---
```

## Legende

- 🚫 **BLOCKED** - Anfrage wurde blockiert
- ⚠️ **WARNING** - Anfrage wurde erlaubt, aber protokolliert
- ✅ **ALLOWED** - Anfrage war unbedenklich
