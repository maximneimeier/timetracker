### Lösung (Pendend)

```bash

  --label-add "traefik.enable=true" \
  --label-add "traefik.http.routers.openclaw.rule=Host(\`srv1480381.hstgr.cloud\`)" \
  --label-add "traefik.http.routers.openclaw.entrypoints=websecure" \
  --label-add "traefik.http.routers.openclaw.tls=true" \
  --label-add "traefik.http.services.openclaw.loadbalancer.server.port=54020" \
  openclaw-ncty-openclaw-1
```
