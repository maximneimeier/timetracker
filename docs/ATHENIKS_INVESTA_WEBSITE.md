## S4 – FEATURES

**H2 DE:** Alles, was du brauchst, um mit Zahlen zu überzeugen.  
**H2 EN:** Everything you need to convince with numbers.

### Feature 1: Ein Modell. Alles synchron.
**DE:** GuV, Cashflow und Bilanz entstehen aus einer einzigen Quelle. Kein manueller Abgleich, keine veralteten Versionen.  
**EN:** P&L, cash flow, and balance sheet come from a single source. No manual reconciliation, no outdated versions.

### Feature 2: Kapitalstruktur auf einen Blick vergleichen.
**DE:** Modelliere Eigen- und Fremdkapital parallel. Tilgung, Zinslast und Kapitalstruktur sind sofort vergleichbar.  
**EN:** Model equity and debt in parallel. Repayment, interest burden, and capital structure are instantly comparable.

### Feature 3: Monatlich und jährlich. Immer konsistent.
**DE:** Beide Planungsebenen laufen automatisch zusammen. Kein manuelles Umrechnen, kein Versionenchaos.  
**EN:** Both planning levels sync automatically. No manual conversion, no version chaos.

## Zusammenfassung Investa

| Aspekt | Beschreibung |
|--------|--------------|
| **Zielgruppe** | CFOs, Geschäftsführer, Investoren |
| **Use Case** | Große Investitionsentscheidungen (nicht Projekte, sondern strategische Investitionen) |
| **Pain Point** | Mehrere Excel-Tabellen, keine Konsistenz, Glaubwürdigkeitsverlust im Board |
| **Lösung** | Ein integriertes Modell für GuV, Cashflow, Bilanz |
| **Key Features** | Echtzeit-Berechnung, Kapitalstruktur-Vergleich, monatlich + jährlich synchron |
| **Time-to-Value** | 20 Minuten bis zum laufenden Modell |
| **Onboarding** | Kein Setup, keine Kreditkarte, sofort startklar |

## Unterscheidung zu Calculus

| | **Investa** | **Calculus** |
|---|---|---|
| **Fokus** | Strategische Investitionsentscheidungen | Projekte & Einzelinvestitionen |
| **Komplexität** | Hoch (GuV, Bilanz, Cashflow integriert) | Mittel (Projekt-Cashflows) |
| **Zielgruppe** | Board-Entscheidungen, Investoren | Projektleiter, CFOs |
| **Output** | Vollständiges Finanzmodell | Projekt-KPIs (NPV, ROI) |
| **Zeitrahmen** | Langfristig (Jahre) | Projektdauer (Monate/Jahre) |
