# Investa Release Workflow

## Übersicht
Drei Produktseiten für Planer, Calculus & Investa – jede mit Hero, Problem, Lösung, Features, Formular.

### Seitenstruktur (pro Seite)
1. **Hero-Bereich** – Eyebrow, Headline, 2 CTAs
2. **Problem-Section** – drei Pain Points mit Icons
3. **Lösung-Section** – Produkt-Vorteile
4. **Features** – 4 Feature-Cards
5. **Formular** – Email + Produkt-Auswahl + Absenden

### Gemeinsamer Workflow
- Nutzer füllt Formular aus (Email, Produkt, Rolle)
- System legt Kontakt in CRM + Notion + Mailchimp an
- Bestätigungsmail wird versendet
- Atheniks-Team wird benachrichtigt

### Design
- Tailwind CSS (wie bestehende Seiten)
- Atheniks Branding (Blau #1e3a8a, Weiß)
- Responsive, modern, sauber

### Integration
- Notion-Datenbank für Kontakte
- Mailchimp/SendGrid für Email-Benachrichtigungen

**Zeitaufwand:** 3x Seiten × 10 Min = 30 Minuten Gesamt
