1. MAX. 2 ORDNEREBENEN

2. KEINE "SONSTIGES"-ORDNER
   Entweder passt es in eine Kategorie oder es wird gelöscht

3. NAMENSKONVENTION
   - Keine Leerzeichen (Unterstriche stattdessen)
   - Keine Sonderzeichen (nur A-Z, a-z, 0-9, _)
   - Datumsformat: YYYY-MM-DD (ISO 8601)

4. FREITAG = AUFRÄUM-TAG
   - 99_TEMP leeren
   - Fertige Projekte von 00_AKTUELL → 04_Archiv/2026/

5. SENSIBLE DATEN
   - Vertraege und Finanzen nur in 01_Atheniks_Firma/
   - Keine sensiblen Daten in 99_TEMP lassen
