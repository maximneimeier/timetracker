# Atheniks Proviso

**Status:** Internes Produkt-Overview · Confidential · 2026  
**Zielgruppe:** Finance Teams, Controller, Steuerberater  
**Kernfunktion:** Automatisierte Vertrags-Rückstellungen (Accruals)

---

## Was ist Proviso?

Eine **Finance-Automatisierungsplattform**, die Verträge in verifizierte, DATEV-fertige Rückstellungs-Buchungspakete umwandelt — jeden Monat automatisch.

**Tagline:** *Contract-Based Accruals — Automated.*

---

## Das Problem: Manuelle Rückstellungen sind fehleranfällig

### Heutiger Workflow (Manuell)
1. **Excel öffnen** — Vertragsliste aus Shared Drive ziehen (hoffentlich aktuell)
2. **Manuell berechnen** — Formeln für jeden Vertragstyp (unterschiedliche Logik)
3. **Mit Ist-Werten abgleichen** — Abweichungen per Hand korrigieren
4. **Mit Vormonat abstimmen** — Änderungen? Verlängerung? Preisanpassung? Alles manuell
5. **In DATEV eintippen** — Jeder Buchungssatz einzeln eingeben
6. **An Steuerberater senden** — PDF/Excel per Mail, Rückfragen, Fehler beheben

**Zeitaufwand:** 2-4 Stunden pro Monat pro Controller  
**Probleme:** Fehleranfällig, keine Audit-Trail, Version-Chaos

---

## Die Lösung: Proviso Workflow (Automatisiert)

```
┌─────────────────────────────────────────┐
│  1. Verträge erfassen                    │
│     Import oder Eingabe von Lieferanten- │
│     und Dienstleistungsverträgen         │
├─────────────────────────────────────────┤
│  2. Auto-Berechnung                      │
│     Proviso berechnet monatliche           │
│     Rückstellungen aus Vertragslogik     │
├─────────────────────────────────────────┤
│  3. Prüfen & Freigeben                   │
│     Controller prüft, Anomalien werden   │
│     markiert, Freigabe mit einem Klick   │
│     Status: Offen → Geprüft → Gebucht    │
├─────────────────────────────────────────┤
│  4. Export nach DATEV                    │
│     Fertiges Buchungspaket wird          │
│     generiert — keine manuelle Eingabe   │
└─────────────────────────────────────────┘
```

**Zeitaufwand:** 15 Minuten pro Monat  
**Vorteile:** Null Fehler, vollständiger Audit-Trail

## Kernfunktionen

| Funktion | Beschreibung |
|----------|--------------|
| **Vertragserfassung** | Import oder manuelle Eingabe aller Verträge |
| **Auto-Berechnung** | Monatliche Rückstellungen automatisch am 1. des Monats |
| **Dashboard & Review** | Anomalien werden automatisch markiert, Übersicht auf einem Screen |
| **Freigabe-Workflow** | Ein-Klick-Freigabe mit Status-Tracking |
| **DATEV-Export** | Sofort generiertes, fertiges Buchungspaket |
| **Audit-Trail** | Vollständige Nachvollziehbarkeit aller Änderungen |
