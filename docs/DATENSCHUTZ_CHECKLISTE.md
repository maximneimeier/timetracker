# DSGVO-Checkliste für Atheniks Registrierungsseite

## Was aktuell fehlt (Stand: 7. Mai 2026)

### 1. Checkboxen im Registrierungsformular
- [ ] AGB-Zustimmung (Pflicht)
- [ ] Datenschutzerklärung zur Kenntnis genommen (Pflicht)
- [ ] Marketing-Einwilligung (optional, NICHT vorausgewählt)

### 2. Links/Texte die existieren müssen
- [ ] AGB-Seite (Text)
- [ ] Datenschutzerklärung (Text)
- [ ] Impressum (Text)

### 3. Technische Anforderungen
- [ ] Button erst aktiv nach Pflicht-Checkboxes
- [ ] Keine vorausgewählten Checkboxen
- [ ] Datenschutz-Link öffnet sich in neuem Tab oder Modal
- [ ] Impressum im Footer erreichbar
