REGELN (WICHTIG!)
================================================================================

1. MAX. 2 ORDNEREBENEN
   ✅ OK: 01_Atheniks_Firma/Investoren/Pitch_Decks/
   ❌ NICHT: 01_Atheniks_Firma/Investoren/Pitch_Decks/2026/Q1/...

2. KEINE "SONSTIGES"-ORDNER
   Entweder eine Kategorie passt, oder die Datei wird gelöscht.

3. NAMENSKONVENTION
   - Keine Leerzeichen → Unterstriche
   - Datum immer: YYYY-MM-DD
   - Beispiel: Vertrag_DesignOffices_2026-04-01.pdf

4. FREITAG = AUFRÄUM-TAG
   - 99_TEMP leeren
   - Fertige Projekte von 00_AKTUELL → 04_Archiv

5. PRIVAT vs. GESCHÄFTLICH
   - Private Fotos/Urlaub: AUF KEINEN FALL im Business-OneDrive!
   - Persönliche Finanzen: Nur wenn direkt mit Firma verbunden

6. CODE & TECHNIK
   - Source Code: GitHub/GitLab (NICHT OneDrive!)
   - Code-Doku: 01_Atheniks_Firma/Produkt_Doku/Technische_Doku/
